# Introduction

OpenBridge Web Components is part of the OpenBridge Design System, an open-source framework for designing and implementing high-performance user interfaces for industrial and maritime workplaces. The system is based on modern UI principles, industry regulations, and human-centered design practices, and is developed in collaboration with leading industry and academic partners.

The OpenBridge Design System provides a comprehensive set of guidelines, components, and tools to ensure consistency, usability, and safety in complex environments. Originally created for maritime use cases, it now supports a wide range of industrial applications.

Learn more about the OpenBridge Design System at [openbridge.no](https://www.openbridge.no/).
