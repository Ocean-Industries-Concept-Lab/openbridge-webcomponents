# Joint Industry Project (JIP)

The OpenBridge Web Components library is developed as part of a Joint Industry Project (JIP), bringing together industry partners and academic institutions to advance the design and implementation of user interfaces for industrial and maritime workplaces.

## What is the JIP?

The JIP is a collaborative initiative where partners funds the implementation of the OpenBridge Web Components library. The project aims to create open, standardized solutions for high-performance workplaces.

## How to Join

Access to the codebase is currently limited to JIP partners and donors until the public release in March 2026. If you or your organization would like to participate, contribute, or gain early access, you can register your interest using [this form](https://docs.google.com/forms/d/e/1FAIpQLSd2H7bbL_duBTMhHzjw7W52H9XXAiJ9A3sL7PsrfCTW_bNUhw/viewform) or read more about the project [here](https://docs.google.com/document/d/18ytBiUrfQrMYOPPz-hd7pgPjnG8ZBG-zr9xYl5Y2TTs/edit?tab=t.0).
